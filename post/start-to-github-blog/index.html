开始用Hugo和Github写博客(POST)

配置步骤

  1. Install Homebrew

  2. brew install go

  3. brew install hugo

  4. 建自己的博客文件夹

  5. 注册Github

  6. 建“（你的用户名）.github.io”的项目

  7. 安装Github Desktop应用

  8. 用Github Desktop来同步自己建好的本地博客文件夹到Github上。
