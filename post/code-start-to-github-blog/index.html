多年经验｜如何用Hugo＋Github写博客

By maxsou

2016年03月09日

准备工作

1.安装brew 


$ Install Homebrew

2.用brew安装go语言 


$ brew install go

3.用brew安装hugo 


$ brew install hugo

建自己的博客

  1. 建自己的博客文件夹

  2. 注册Github

  3. 建“（你的用户名）.github.io”的项目

  4. 安装Github Desktop应用

  5. 用Github Desktop来同步自己建好的本地博客文件夹到Github上。

更换主题

看到喜欢的主题（theme）可以下载到自己的博客相应文件夹“theme”或者“themes”里，

然后在config.toml里修改配置“theme＝“xxxx””即可。
