多年经验｜如何用Hugo＋Github写博客

配置步骤

1.安装brew


$ Install Homebrew

2.用brew安装go语言


$ brew install go

3.用brew安装hugo


$ brew install hugo

  1. 建自己的博客文件夹

  2. 注册Github

  3. 建“（你的用户名）.github.io”的项目

  4. 安装Github Desktop应用

  5. 用Github Desktop来同步自己建好的本地博客文件夹到Github上。
